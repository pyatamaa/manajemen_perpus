# manajemen_perpus
ini adalah repository untuk kode manajemen perpustakaan sederhana berbasis website, dibuat menggunakan bahasa PHP ver 8.1, bootstrap 5.3.3, dan database MySQL.

cara import database ke MySQL :

1. buat terlebih dahulu database dengan nama **manajemen_perpustakaan**

2. lalu import file sql **manajemen_perpustakaan** kedalam database yang sudah anda buat
